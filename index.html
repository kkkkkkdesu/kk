千と千尋の神隠し

千と千尋の神隠しは、宮崎駿監督によるジブリの名作アニメーション映画です。

このウェブサイトは、その世界観を再現したものです。

body { font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif; margin: 0; padding: 0; background-color: #f0f0f0; overflow-x: hidden; /* 横スクロールを禁止 */ } .main { max-width: 800px; margin: 0 auto; padding: 20px; } header { text-align: center; } h1 { font-size: 36px; color: #333; } .content { background-color: #fff; padding: 20px; margin-top: 20px; border-radius: 8px; box-shadow: 0 0 10px rgba(0, 0, 0, 0.1); } /* 千と千尋の神隠しの背景 */ .background { position: fixed; top: 0; left: 0; width: 100%; height: 100%; background-image: url('https://i.imgur.com/lCwZ6Zq.jpg'); background-size: cover; z-index: -1; opacity: 0.5; /* 背景画像の透明度 */ animation: backgroundFade 20s infinite alternate; /* 背景のアニメーション */ } /* キャラクター */ .character { position: fixed; bottom: 0; left: 50%; transform: translateX(-50%); width: 200px; z-index: 1; animation: characterAppear 10s forwards; /* キャラクターのアニメーション */ opacity: 0; /* 初期は非表示 */ } @keyframes backgroundFade { 0% { opacity: 0.5; } 50% { opacity: 0.3; } 100% { opacity: 0.5; } } @keyframes characterAppear { 0% { opacity: 0; bottom: -200px; /* 下からスライドイン */ } 100% { opacity: 1; bottom: 20px; } } // 背景の追加 const background = document.createElement('div'); background.classList.add('background'); document.body.appendChild(background); // キャラクターの追加 const character = document.createElement('img'); character.src = 'https://i.imgur.com/txbZs4T.png'; // キャラクターの画像 character.alt = '千と千尋'; character.classList.add('character'); document.body.appendChild(character);
